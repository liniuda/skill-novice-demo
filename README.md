# Skill Market — CCO Skill Studio

> 智服工坊：CCO 客服技能集市 & AI 拟人化配置中心（纯前端 Demo）

在线访问：通过 GitHub Pages 部署，进入 **Work → 拟人化中心** 体验。

## 功能模块

- **Skill 配置**：技能元数据、Prompt、知识库、工具、灰度
- **拟人化中心**：A 人设档案 / B 记忆与画像 / C 知识与意图 / D 风格与解码 / E 节奏与互动 / F 安全与改写 / G 兜底与转人工
- **发布管理**：版本/审批/灰度策略
- **MCP 服务**：MCP toolkit 管理
- **Market**：技能广场 / 发布中心 / 我的发布

## 技术栈

- 原生 ES Module + localStorage 持久化
- Apple HIG 风格设计 / 4pt grid / 阿里橙主题

## 本地运行

```bash
python3 -m http.server 5173
# 打开 http://localhost:5173
```
